KDE Meetup

  • Meetup
Event date: 
Sat, 23/02/2013 - 10:00 - Sun, 24/02/2013 - 18:00

KDE Meetup is an event for Qt/KDE contributors and open source enthusiasts to meet up, share their knowledge, contribute, learn, play, have fun and create limitless possibilities using Qt and KDE. KDE Meetup is the first large scale open source Meetup in Gujarat and the largest meetup of KDE contributors in India since conf.kde.in in 2011. It will run for 2 days with talks on the latest KDE developments, sessions on how you can start contributing to open source, coding sessions and workshop.

http://www.gdgdaiict.com/kdemeetup/

Event date: 
Fri, 22/02/2013 - 17:00 - Sun, 24/02/2013 - 18:00

FOSSMeet is an annual event at NIT Calicut that brings together the Free and Open Source Community from around the country. It was started in 2005 to encourage contribution to free and open source software. The last 7 editions were hugely successful and brought together a lot of enthusiasts.

Upholding the philosophy of FOSS, the event will have talks and workshops dealing with the current trends in FOSS and open hardware, hands-on sessions to develop skills and a place to meet like minded people.

Visit http://fossmeet.in for more details.

Python has several libraries that enable numeric and symbolic computing. It also interfaces well with native codes in C,C++ and FORTRAN and has excellent libraries for plotting/visualization. Python also provides libraries for general purpose computing including generating user interfaces and web development. These coupled with it's interactive interpreter make Python an excellent language for Scientific Computing.
